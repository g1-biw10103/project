2. Layout and Positioning

Learn the basics of CSS including selectors, properties and how to apply styles to HTML elements.By the end, you will be able to design complex page structures that adapt to different screen sizes.

Category:CSS Layout Techniques

Instructor: Ah Mei

Rating: 4.8/5.0

Credit Hours: 4

Module 1: Introduction to CSS Layout

This module introduces the concept of layout in CSS. Learners will understand how web pages are structured and how CSS controls the positioning and arrangement of elements on a webpage.

Preview

Module 2: CSS Positioning

This module focuses on CSS positioning methods such as static, relative, absolute, fixed, and sticky. Learners will understand how elements move and behave on a page using different position values.

Preview

Module 3: CSS Flexbox Layout

This module introduces Flexbox, a one-dimensional layout system in CSS. Learners will learn how to align, space, and arrange elements efficiently in rows or columns.

Preview

Module 4: CSS Grid Layout

This module introduces CSS Grid, a two-dimensional layout system. Learners will learn how to create complex and responsive layouts using rows and columns.
