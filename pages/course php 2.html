PHP For Beginners

2. Working with Databases

Connect to MySQL databases and perform CRUD operations. This helps you store, retrieve, and manage data efficiently in your web projects.

Category: PHP / Databases

Instructor: Syainun

Rating: 4.6/5.0

Credit Hours: 4

Module 1: Introduction to Databases & MySQL

This module introduces relational databases and MySQL. Learners will understand database concepts such as tables, records, primary keys, and foreign keys, and learn how PHP interacts with MySQL to store and retrieve data.

Preview

Module 2: Connecting PHP with MySQL

Learn how to declare and use variables in PHP. This module covers data types such as strings, integers, floats, booleans, arrays, and how to store and manipulate information in PHP scripts.

Preview

Module 3:CRUD Operations

This module teaches how to perform CRUD operations with PHP and MySQL. Learners will practice inserting data, retrieving records, updating data, and deleting records from a database.

Preview

Module 4: Prepared Statements & Security

Learn how to use prepared statements to prevent SQL injection attacks. This module also covers basic security practices when working with PHP and MySQL, such as input validation and password hashing.
