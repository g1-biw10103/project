PHP For Beginners

1.PHP Fundamentals

Learn syntax, variables and basic programming concepts in PHP.You will also learn how to write dynamic server side scripts to power web applications.

Category: Frameworks and Libraries

Instructor: Syahirah

Rating: 4.8/5.0

Credit Hours: 4

Module 1: Introduction to PHP

This module introduces PHP, a popular server-side scripting language used to create dynamic web pages. Learners will understand PHP syntax, how it integrates with HTML, and basic setup for running PHP scripts on a local server.

Preview

Module 2: PHP Variables & Data Types

Learn how to declare and use variables in PHP. This module covers data types such as strings, integers, floats, booleans, arrays, and how to store and manipulate information in PHP scripts.

Preview

Module 3:PHP Functions & Operators

This module teaches how to create and use functions in PHP, including passing arguments, returning values, and using built-in PHP functions. Learners will also understand common operators for arithmetic, comparison, and logical operations.

Preview

Module 4: PHP Control Structures

Learn how to use control structures in PHP, including conditional statements (if, else, switch) and loops (for, while, foreach). This module helps you write dynamic scripts that can make decisions and repeat actions based on conditions.
