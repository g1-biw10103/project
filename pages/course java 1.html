JavaScript For Beginners

1. JavaScript Basics

Learn variables, functions and control structures in JavaScript.You will also learn how to write simple scripts to make web pages interactive..

Category: Fundamentals JavaScript

Instructor: Aisyah zila

Rating: 4.9/5.0

Credit Hours: 4

Module 1: Introduction to JavaScript

This module introduces JavaScript, the programming language of the web. Learners will understand what JavaScript is how it works with HTML and CSS and how to include scripts in a web page to add interactivity.

Preview

Module 2: Variables & Data Types

Learners will explore variables in JavaScript, including var, let, and const. The module also covers basic data types such as strings, numbers, booleans, arrays and objects and how to store and manipulate data.

Preview

Module 3: Functions & Scope

This module covers functions in JavaScript, including how to declare and call functions, pass arguments, return values and understand variable scope

Preview

Module 4: Control Structures

Learners will understand control flow in JavaScript using conditional statements (if, else if, else), loops (for, while, for...of) and switch statements to execute code based on conditions or repeat tasks.
