Cascading Style Sheets For Beginners

2.

Learn the basics of CSS including selectors, properties and how to apply styles to HTML elements. You will also understand how CSS interacts with HTML to create visually appealing web pages.

Category:Web Page Layout Design

Instructor: Ah Mei

Rating: 4.7/5.0

Credit Hours: 4

Module 1 : Introduction to CSS

This module introduces Cascading Style Sheets and its role in web development. Learners will understand how CSS is used to style HTML elements, control layout, colors, fonts and improve the visual appearance of websites.

Preview

Module 2: CSS Syntax & Selectors

This module explains the basic structure of CSS and learners will practice using element selectors, class selectors and ID selectors to apply styles to HTML content.

Preview

Module 3: Colors, Fonts & Text Styling

Learners will learn how to change text color, background color, font family, font size, alignment and spacing to make web pages more attractive.

Preview

Module 4: CSS Box Model & Layout Basics

This module introduces the CSS Box Model including margin, border, padding and content. Learners will also understand basic layout concepts used to structure web pages properly.
