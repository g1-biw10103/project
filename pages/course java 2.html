JavaScript For Beginners

2. DOM Manipulation

Interact with HTML elements using the Document Object Model.This allows you to dynamically update content, styles and structure on your web pages.

Category: DOM Manipulation

Instructor: Akamal Syahin

Rating: 4.7/5.0

Credit Hours: 4

Module 1: Introduction to JavaScript

This module introduces the Document Object Model (DOM), which represents the structure of a web page. Learners will understand how HTML elements are structured as objects that can be accessed and manipulated using JavaScript.

Preview

Module 2: Selecting DOM Elements

Learn how to access DOM elements using methods like getElementById, getElementsByClassName, querySelector, and querySelectorAll. This is essential for manipulating page content dynamically.

Preview

Module 3: Modifying DOM Elements

This module teaches how to change content, attributes, and styles of DOM elements using JavaScript. Learners will practice updating text, images, classes and inline styles dynamically.

Preview

Module 4: Handling DOM Events

Learn how to make web pages interactive by handling events such as click, submit, mouseover, and keyboard events. This module covers adding event listeners and responding to user actions.
